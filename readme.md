A fitness tracker app

The first version of LogBuddy is complete but even I don't even use it. Why?

Rough edges makes it awful to use (ex: keyboard covering inputs):
- It's not local first, must create an account before usage (dealbreaker)
- Trying to do too many things at once (tracking 5 different completely different things just makes for a worse overall experience). Focus is needed.
- Workout templates are impractical. In practice you want to start workouts based off of past workouts and modify workouts on the fly. So really, the base datum isn't the workout, it's the exercise. Exercises are stateful, while workouts are transitory
- Not already having a food database to choose from is annoying
- Looked like shit, felt slow and clunky
- Not a fan of react or ionic, I prefer static typing
- This new version of LogBuddy is designed to fixed all the issues of the old version. Still under active development....
